Tic-Tac-Toe the Hard Way
========================

The exercises in this 'Tic-Tac-Toe the Hard Way' exercise are build around this podcast series `Tic-Tac-Toe the Hard Way <https://pair.withgoogle.com/thehardway/>`_


.. image:: _static/Podcast_Cover_small.jpg



You are supposed to listen to the podcasts and create your own code from what they talk about in the podcast series. 

You will not be able to finnish it right away, but the whole exercises series is also meant to be done in smaler steps, and is supposed to talk 3 weeks (or more).
